Linux命令学习

发表于 2019-11-15 | 分类于 linux |

出于个人需要，最近自己在笔记本里装了ubuntu18.04，也涉及到了各种终端操作，所以写一篇文章慢慢的把相关的命令记录一下，也方便自己理解记忆。同时也会慢慢更新。

ls

经常使用的命令，能够查看目标列表，输出信息可以进行彩色加亮显示，来区分不同类型文件。

语法
选项
  • / 显示除开头为”.”的文件及目录 (”/“代表不输入)
  • -a 显示所有文件及目录,包括”.”开头文件及目录
  • -l 将文件所有信息，包括名称、型态、权限、拥有者、文件大小等详细列出
  • -r 将文件以英文反序显示
  • -t 将文件按照建立时间先后次序列出
  • -A 同-a，但不列出 “.” (目前目录) 及 “..” (父目录)
  • -F 在列出的文件名称后加符号，如可执行档则加 “*”, 目录则加 “/“
  • -R 若目录下有文件，则文件也按照次序列出

mv

mv命令取自”move”缩写，通常用来移动文件以及目录，或者重命名文件及目录。

语法
选项
  • -i 若指定目录前已有同名文件，则先询问是否覆盖旧文件
  • -f 在mv操作要覆盖已有目标文件时直接覆盖

cp

cp命令主要用于复制文件或目录。和mv命令的区别在用cp复制时源文件状态不会更改，而mv会修改源文件，相当于windows的剪切功能。

语法
选项
  • -a 在复制目录时使用，保留链接、文件属性、并复制目录下所有内容
  • -d 复制时保留链接，该链接相当于windows中快捷方式。
  • -f 覆盖已存在目标文件而不给提示
  • -i 覆盖前给出提示
  • -p 除复制文件内容外，将修改时间和访问权限复制到新文件中
  • -r 若源文件是一个目录，则复制该目录下所有子目录和文件
  • -l 不复制文件，只生成链接文件

cd

cd命令用于切换当前工作目录至指定目录路径，指定目录路径可以为相对路径，也可以为绝对路径。

语法
选项
  • - 返回进入此目录前所在目录
  • ~ 进入用户主目录
  • .. 返回上层目录
  • / 进入根目录

rm

rm命令用于删除文件或目录，但使用该命令后是不可恢复的，所以要慎重使用。

语法
选项
  • -i 删除前进行逐一确认
  • -f 即使文件设为只读，也直接删除，无需确认
  • -r 删除目录下所有文件
